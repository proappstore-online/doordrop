## Where data lives

Everything you save in Doordrop is stored on Cloudflare infrastructure
(D1, R2, KV) under your ProAppStore account. The app developer does not
have access to your personal data — only daily aggregate counts (active
users, session-minutes, API calls) used to size their payout share.

Doordrop uses the same session cookie as the rest of ProAppStore for

## Data deletion

Delete your account from [dashboard.proappstore.online](https://dashboard.proappstore.online).
That removes all data Doordrop stored for you, alongside the
platform-level deletion described in the [platform privacy policy](https://proappstore.online/privacy#data-deletion).
